# StaySense-AI-Airbnb-Analysis
An end-to-end machine learning project for analyzing Airbnb NYC listings and predicting prices using data preprocessing, exploratory data analysis (EDA), and regression models.

# StaySense AI: Intelligent Airbnb Price & Booking Pattern Analysis Using Machine Learning

## Project Overview
This project analyzes the Airbnb NYC 2019 dataset to discover trends in listing prices, room types, neighborhoods, reviews, and availability. It also uses machine learning models to predict Airbnb listing prices.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Price Prediction using Machine Learning
- Model Performance Evaluation

## Dataset
Airbnb NYC 2019 Dataset
